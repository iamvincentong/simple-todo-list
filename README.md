## Introduction
This is a simple todo list application powered by Laravel, Sanctum and Socialite.

## Github
Clone the repo from github: https://github.com/rising728/simple-todo-list.git

## Docker Compose
Go to the project folder and run docker command:
````
docker-compose up
````

## Route
Login with Github account using the following route:

http://localhost:8000/api/login/github

## API Endpoint
http://localhost:8000/api/tasks
